# 任务管理系统

## 项目简介
本项目是一个任务管理系统，支持用户通过三种不同的数据结构（顺序表、单向链表、双向链表）实现任务的增删改查、排序、保存和加载功能。用户可以根据需求选择不同的数据结构实现。

## 项目结构
```
Lab1/
├── DoublyLinkedTaskManager.cpp   // 双向链表实现的任务管理器
├── DoublyLinkedTaskManager.h     // 双向链表实现的任务管理器头文件
├── LinkedTaskManager.cpp         // 单向链表实现的任务管理器
├── LinkedTaskManager.h           // 单向链表实现的任务管理器头文件
├── SequentialTaskManager.cpp     // 顺序表实现的任务管理器
├── SequentialTaskManager.h       // 顺序表实现的任务管理器头文件
├── Task.h                        // 定义任务数据结构
├── TaskManager.h                 // 抽象任务管理器基类
├── main.cpp                      // 主程序入口，提供用户交互界面
├── tasks_10000.txt               // 示例任务数据文件
└── 实验1线性表的应用.docx        // 实验文档
```

### 文件说明
- **DoublyLinkedTaskManager.cpp / .h**
  - 使用双向链表实现任务管理器。
  - 支持高效的任务插入、删除操作，尤其适合需要频繁在链表两端操作的场景。

- **LinkedTaskManager.cpp / .h**
  - 使用单向链表实现任务管理器。
  - 实现简单，适合任务量较小且操作集中在链表尾部的场景。

- **SequentialTaskManager.cpp / .h**
  - 使用顺序表（`std::vector`）实现任务管理器。
  - 适合任务量较小且查询操作频繁的场景。

- **Task.h**
  - 定义了任务的基本数据结构，包括任务名称、描述、优先级和截止日期。

- **TaskManager.h**
  - 定义了任务管理器的抽象基类，包含所有任务管理功能的纯虚函数。

- **main.cpp**
  - 提供用户交互界面，允许用户选择数据结构并执行任务管理操作。

- **tasks_10000.txt**
  - 示例任务数据文件，可用于测试任务加载功能。

- **实验1线性表的应用.docx**
  - 实验文档，描述了实验的背景、目标和要求。

## 三种数据结构实现方法的比较

| 特性                | 顺序表（`std::vector`） | 单向链表             | 双向链表             |
|---------------------|-------------------------|----------------------|----------------------|
| **插入效率**       | 较低（尾部高效，其他位置低效） | 较高（尾部高效）     | 高（头尾均高效）     |
| **删除效率**       | 较低（需移动元素）     | 较高（需遍历找到元素）| 高（头尾均高效）     |
| **查询效率**       | 高（随机访问）         | 较低（需遍历）       | 较低（需遍历）       |
| **内存使用**       | 连续内存，可能浪费空间 | 分散内存，节省空间   | 分散内存，额外存储前驱指针 |
| **实现复杂度**     | 简单                   | 较简单               | 较复杂               |

### 适用场景
- **顺序表**：适合任务量较小且查询操作频繁的场景。
- **单向链表**：适合任务量中等且操作集中在链表尾部的场景。
- **双向链表**：适合任务量较大且需要频繁在链表两端操作的场景。

## 使用说明
1. 编译项目：
   ```bash
   g++ -o TaskManager main.cpp SequentialTaskManager.cpp LinkedTaskManager.cpp DoublyLinkedTaskManager.cpp -std=c++11
   ```

2. 运行程序：
   ```bash
   ./TaskManager
   ```

3. 根据提示选择数据结构并执行任务管理操作。
